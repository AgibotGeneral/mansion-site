MANSION is a project for **multi-floor, long-horizon embodied tasks** where agents must operate across
**multiple floors**, use **stairs/elevators**, and reason about object state and placement over time.

This website contains:

- **Getting started**: install + run your first episode
- **Core concepts**: multi-floor world model, task format, and assumptions
- **Dataset**: MansionWorld (scenes, assets, annotations)
- **Environment**: observations, metadata, object state, and scene loading
- **Skills & actions**: a high-level API for long-horizon planning
- **API reference**: stable interfaces you can build on

:::tip
If you like iTHOR’s *API reference* style, jump directly to **API → Overview**.
:::
