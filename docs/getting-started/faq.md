### Does MANSION require AI2-THOR?
Not necessarily. MANSION is a *project layer* that can wrap a simulator backend. If you currently
use AI2-THOR, you can integrate MANSION skills and multi-floor tracking on top.

### Do I need to use the high-level skills?
No — but they are recommended for long-horizon tasks to keep task code clean and consistent.

### Where do scene IDs come from?
See **Dataset → MansionWorld**. Scene IDs typically map to JSON scene specs + asset bundles.

### How do I cite this project?
See **Project → Citation**.
